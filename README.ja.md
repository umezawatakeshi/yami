# YAMI - Yet-another Auction Manager for Intranet

**YAMI** は組織内ネットワーク（つまりイントラネット）でオークションを開催するためのシンプルな Web アプリケーションです。

## やることとやらないこと

### やること

- オークションの出品と入札の調停

### やらないこと

- ユーザー管理
